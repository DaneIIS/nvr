Exadel CompreFace is a free and open-source face recognition service.

It provides capabilities for:

- Face recognition
- Face verification
- Face detection
- Landmark detection
- Mask detection
- Head pose detection
- Age
- Gender recognition

See the [getting started guide](https://github.com/exadel-inc/CompreFace#getting-started-with-compreface) to learn how to use CompreFace.

## Configuration

<details>
  <summary>Configuration example</summary>

```yaml title="/config/config.yaml"
compreface:
  face_recognition:
    host: compreface
    port: 8000
    recognition_api_key: !secret compreface_api_key
    train: true
    face_plugins: age,gender
    cameras:
      camera_one:
    labels:
      - person
```
